## Webhook handling checklist

1. Receive events on a dedicated backend endpoint.
2. Verify authenticity before processing payloads.
3. Return a success response quickly.
4. Process business logic asynchronously when possible.

## Reliability guidance

- Support retries and deduplicate repeated deliveries.
- Capture event IDs for observability and replay workflows.
- Alert on repeated delivery failures or signature validation errors.
