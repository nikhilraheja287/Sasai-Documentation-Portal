---
title: How do I get client secret?
description: Retrieve your client secret from the dashboard and store it safely for backend use.
---

## Where to find it

1. Open your project in the Sasai Dashboard.
2. Go to the developer credentials or application settings area.
3. Reveal or regenerate the client secret if your role allows it.

## Store it safely

- Copy it into a secure secrets manager or deployment environment variable.
- Limit access to the smallest number of operators and services.
- Rotate it immediately if it is shared in chat, screenshots, or logs.

<Callout type="warn" title="Shown once in some flows">
  Some dashboards only display a newly created secret once. If you lose it, regenerate a new value instead of searching old messages or local notes.
</Callout>
